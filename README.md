
## Project Setup

```sh
npm install
```
## Install Json Server

```sh
npm i json-server
```

## Fire up Json Server

```sh
npm run db-serve 
```
### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
